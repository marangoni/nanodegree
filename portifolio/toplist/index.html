HTML

Definições

HTML significa - Hyper Text Markup Language , ou linguagem de marcação de hipertextos e é a linguagem utilizada para codificação de páginas web.

A linguagem HTML utiliza os chamados elementos que informam ao navegador como o conteúdo deve ser exibido.

Elementos

Os elementos da linguagem HTML são formados por caracteres mnemônicos delimitados pelos símbolos < e >, denominados tags.

Tais elementos podem ser compostos por somente tags de abertura ou tags de abertura e fechamento, como veremos em alguns exemplos a seguir:

Exemplos:

Elementos com tags de abertura e fechamento:

    Tag - < b > e < /b > - bold - Mostra o texto em negrito.
    Tag - < em > e < /em > - emphasise - Mostra o texto em itálico.

Elementos somente com tag de abertura:

    Tag - < br > break - Quebra de linha.
    Tag - < hr > horizontal row - Insere linha horizontal.

Elementos in line e em bloco

Os elementos da linguagem HTML podem ainda ser em bloco ou alinhados. Dos elementos estudados até o momento são alinhadas ( inline ) as tags: a, spam, br, img e strong e são em bloco as tags div e p.

Atributos

Todos os elementos HTML podem ter atributos que fornecem informação adicional sobre o elemento. Os atributos são sempre especificados na tag de início.

Atributos geralmente seguem o padrão nome/valor .

Exemplos:

    < a href="https://www.w3schools.com"> Este é um link < /a>
    < img src="img_girl.jpg">
